Research Data
MCP Servers

Bring-your-own-key MCP endpoints for DataTagger (FDM) and elabFTW. Register your API credentials and connect your AI editor in one step.

No accounts, no cloud, no middleware — just proxy routes under a single domain.

How to Use

1

Register

Click any card below, paste your API key / FDM token, and get a session URL back.

2

Copy the URL

The page gives you a URL like https://researchmcp.duckdns.org/el/mcp?token=...

3

Paste in your agent

Add it to your MCP client config (Hermes, Claude Desktop, etc.) and connect.

Hermes Agent config example 
mcp_servers:
  elabmcp:
    url: "https://researchmcp.duckdns.org/el/mcp?token=[paste-here]"
    timeout: 120
    connect_timeout: 30
  datatagger:
    url: "https://researchmcp.duckdns.org/dt/mcp?token=[paste-here]"
    timeout: 120
    connect_timeout: 30

Then type /reload-mcp in Hermes CLI — tools appear as mcp_elabmcp_* and mcp_datatagger_*.
